01 / Over Mij

Ik geloof dat goede code en goed design dezelfde discipline zijn.

Ik begon met bouwen voor het web omdat ik gefascineerd was door hoe een paar regels code iets konden produceren dat iemand ziet, voelt en waarmee interactie heeft. Die fascinatie is niet verdwenen — ze is verscherpt.

Vandaag werk ik op het snijvlak van design systems en product engineering. Ik geef om de dingen die makkelijk te missen zijn: laadprestaties, toetsenbordnavigatie, de 12ms vertraging die een animatie verkeerd doet aanvoelen.

Buiten code word ik aangetrokken door architectuur, typografie en alles wat met een ongewoon niveau van zorg is gemaakt.

04 / Hoe Ik Werk

Proces is hoe goede intenties goede software worden.

01

Ontdekken

Ik begin met luisteren. Het probleem, de beperkingen en de betrokken mensen begrijpen voordat ik een tekstbewerker aanraak. Goede vragen besparen meer tijd dan snel typen.

02

Definiëren

Ik breng de oplossingsarchitectuur, componenthiërarchie en interactieflows in kaart voordat ik productiecode schrijf. Een duidelijk plan is geen overhead — het is snelheid.

03

Bouwen

Ik werk in korte iteraties en lever werkende software vroeg en vaak op. Ik geloof in code review, pair programming en de waarde van een tweede paar ogen.

04

Verfijnen

Ik ben geobsedeerd door de details — prestaties, toegankelijkheid en de micro-interacties die een interface levendig laten aanvoelen. Vakmanschap is wat goed van onvergetelijk scheidt.

05 / Momenteel Aan Het Leren

Ik wil nooit stoppen met nieuwsgierig zijn.

WebGL & Three.js

3D-graphics en shaderprogrammering voor interactieve webervaringen die verder gaan dan het platte scherm.

Motion Design

Geavanceerde animatieprincipes — veerfysica, gechoreografeerde sequenties en de kunst van timing.

Rust

Systeemprogrammering om webprestaties op een dieper niveau te begrijpen. Compileren naar WASM voor de browser.

TypeScript Patterns

Geavanceerde generieke types, template literal types en het bouwen van API's die onmogelijk verkeerd te gebruiken zijn.

Accessibility

WCAG 2.2, ARIA-patronen en testen in de echte wereld met schermlezers. Bouwen voor iedereen.

Design Tokens

Theming op schaal met Style Dictionary, multi-brand design systems en token-architectuur.
