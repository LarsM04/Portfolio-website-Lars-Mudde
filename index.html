01 / Over Mij

Ik geloof dat goede code en goed design dezelfde discipline zijn.

Ik begon met bouwen voor het web omdat ik gefascineerd was door hoe een paar regels code iets konden produceren dat iemand ziet, voelt en waarmee interactie heeft. Die fascinatie is niet verdwenen — ze is verscherpt.

Vandaag werk ik op het snijvlak van design systems en product engineering. Ik geef om de dingen die makkelijk te missen zijn: laadprestaties, toetsenbordnavigatie, de 12ms vertraging die een animatie verkeerd doet aanvoelen.

Buiten code word ik aangetrokken door architectuur, typografie en alles wat met een ongewoon niveau van zorg is gemaakt.
